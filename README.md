## 📖 開啟方式

### 方法 1：直接在瀏覽器開啟（最簡單）
1. 找到下載的 `index.html` 檔案
2. 雙擊檔案，會自動在預設瀏覽器中開啟
3. 開始使用！

### 方法 2：使用本地伺服器（推薦）
如果您想要更穩定的體驗：

```bash
# 如果您有 Python
python -m http.server 8000

# 如果您有 Node.js
npx http-server

# 然後在瀏覽器開啟
http://localhost:8000
```

### 方法 3：部署到網路上（永久訪問）
您可以免費部署到以下平台：

#### Netlify（推薦）
1. 前往 https://app.netlify.com/drop
2. 將 `index.html` 拖曳到頁面
3. 獲得永久網址！

#### GitHub Pages
1. 創建 GitHub 儲存庫
2. 上傳 `index.html`
3. 在設定中啟用 GitHub Pages
4. 訪問 `https://你的帳號.github.io/儲存庫名稱`

#### Vercel
1. 前往 https://vercel.com
2. 匯入專案或拖曳檔案
3. 自動部署完成

## 🎯 功能說明

### 本週任務頁面
- **三個電子報段落**：本週反思/心得、推薦/實踐小物、最近在學/在想
- **進度追蹤**：即時顯示完成百分比
- **完成標記**：點擊按鈕標記段落完成狀態
- **字數統計**：每個段落都有字數顯示
- **一鍵複製**：點擊「複製內容」按鈕，自動整理格式

### 部落格文章頁面
- **標題輸入**：大標題輸入框
- **內容編輯**：寬敞的寫作空間
- **字數追蹤**：目標 500-800 字
- **一鍵複製**：複製標題和內容

### 複製後的工作流程
1. 在系統中完成寫作
2. 點擊「複製內容」
3. 前往 [方格子](https://vocus.cc)
4. 新增文章 → Ctrl+V 貼上
5. 發布！

## 💾 資料儲存

**注意**：目前版本的資料只儲存在瀏覽器中（localStorage）
- 關閉分頁後資料會消失
- 建議隨時點擊「複製內容」備份
- 或者使用「儲存草稿」功能（需要瀏覽器支援）

## 📝 常見問題

### Q: 我的內容會自動儲存嗎？
A: 目前不會。請記得點擊「複製內容」保存您的文字。

### Q: 可以在手機上使用嗎？
A: 可以！但建議使用電腦獲得最佳體驗。

### Q: 如何修改配色或樣式？
A: 可以編輯 `index.html` 中的 style 部分，或聯繫我協助客製化。

### Q: 資料會上傳到哪裡？
A: 完全不會！所有資料都在您的瀏覽器本地，沒有任何上傳。
